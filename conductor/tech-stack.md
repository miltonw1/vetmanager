# Tech Stack

## Language & Core
- **TypeScript:** Primary programming language used across both frontend and backend for type safety and scalability.

## Frontend
- **React (Vite):** Frontend library for building the user interface, utilizing Vite for an optimized development experience and fast builds.
- **Tailwind CSS:** Utility-first CSS framework for efficient and consistent styling.
- **Zustand:** Lightweight state management for handling application state.
- **React Router:** Navigation and routing management.

## Backend
- **NestJS:** Progressive Node.js framework for building the scalable server-side application.
- **Prisma:** Modern database toolkit and ORM used for data modeling and PostgreSQL database interactions.

## Database
- **PostgreSQL:** Relational database management system.

## Quality & Tooling
- **Biome:** Used for fast formatting and linting in both frontend and backend.
- **ESLint:** Additional linting for the React frontend.
- **Jest:** Testing framework for backend unit and integration tests.
- **Cypress:** End-to-end testing framework for frontend verification.
